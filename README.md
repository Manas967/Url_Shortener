## рҹҡҖ Quick Start

### Prerequisites

Make sure you have the following installed:

- **Node.js** `v18+` вҖ” [Download](https://nodejs.org)
- **npm** (comes with Node)
- **MongoDB** вҖ” [Local install](https://www.mongodb.com/try/download/community) or [Atlas (free cloud)](https://www.mongodb.com/atlas)

---

### Step 1 вҖ” Clone the repo

```bash
git clone https://github.com/Manas967/Url_Shortener.git
cd Url_Shortener
```

### Step 2 вҖ” Install dependencies

```bash
npm install
```

### Step 3 вҖ” Create your `.env` file

```bash
# Create the file
touch .env
```

Then open `.env` and add your config (see [Environment Variables](#%EF%B8%8F-environment-variables) below).

### Step 4 вҖ” Start the server

```bash
# Development (auto-restarts on file changes)
npx nodemon server.js

# OR plain Node
node server.js
```

### Step 5 вҖ” Confirm it's running

You should see:
```
Server is running at 5000
MongoDB connected
```

> рҹҺү Your server is live at **`http://localhost:5000`**

### `POST /url/shorten`

Shortens a long URL and returns a unique short ID.

**Request**

```http
POST /url/shorten
Content-Type: application/json
```

```json
{
  "longUrl": "https://www.example.com/some/very/long/url?with=params&and=more"
}
```

**Success Response** вҖ” `201 Created`

```json
{
  "message": "Url shorten successfully",
  "shortId": "aB3xZ9qW"
}
```

**Your short URL is:** `http://localhost:5000/url/aB3xZ9qW`

### Short ID Generator

The `utils/shortId.js` generates an **8-character** string using a mix of:

```
digits      вҶ’  0вҖ“9   (ASCII 48вҖ“57)
uppercase   вҶ’  AвҖ“Z   (ASCII 65вҖ“90)
lowercase   вҶ’  aвҖ“z   (ASCII 97вҖ“122)
```

Each character is picked by randomly choosing one of the three categories, then picking a random ASCII code within that range вҖ” giving a pool of **62 possible characters** and `62вҒё = ~218 billion` unique combinations.
